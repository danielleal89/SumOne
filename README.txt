Tutorial de Utilização da Aplicação

1º - Ao acessar a tela inicial é requerido usuario e senha para se autenticar.
Logins Pré-Cadastrados:
   Usuario: op123
   Senha: op456
ou
   Usuario: master00
   senha: master01

2º - Após é redimencionado para a tela de cadastro de compras, onde contém os campos:
-> CPF(após a primeira inserção o campo é bloqueado), Produto e Valor.
3º - Funcionalidade dos botões:
-> Adicionar(adiciona o produto e o valor a lista)
-> Limpar(limpa os campos para uma nova inserção)
-> Salvar(grava no banco de dados e limpa tudo para um novo cadastro)
-> Voltar(retorna a tela de login)
-> Compras(avança para a tela de acompanhamento das compras cadastradas)

4º - Na tela de compras contém todas as compras cadastradas, com filtros:
-> Por Menor Valor, Operador e Data.
-> Voltar(retorna a tela de cadastro de compras)
